# Linux Commands - Day 1

1. pwd - show current directory 
2.ls - la : list files/ folders with Details
3. cd - change directory 
4. Mkdir - create a new folder
5. Rm - remove folder
6. Cp - copy files/folders
7. Mv - move/rename files/folders
8. Chmod - change file permissions
9. Chown - change file owner
10. Cat - show file contents
11. Less - view file content page by page
12. Nano - edit files in terminal
